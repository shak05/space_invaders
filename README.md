A recreation of the classic Space Invaders arcade game built using Python and Pygame, implementing core gameplay mechanics such as player movement, enemy waves, shooting, collision detection, and scoring.

## Features
- Player-controlled spaceship movement
- Enemy spawning and movement
- Bullet shooting mechanics
- Collision detection between bullets and enemies
- Score tracking system
- Game-over condition
- Classic arcade-style gameplay

##  Tech Stack
- Language: Python
- Library: Pygame

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install pygame
3. Run the game:
   python main.py

##  Purpose
This project was created to practice game development fundamentals, real-time event handling, and collision logic using Pygame while recreating a nostalgic arcade game.
